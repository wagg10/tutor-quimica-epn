### 1. Fundamento químico del problema

La estequiometría permite relacionar las masas de reactivos y productos mediante los coeficientes de una ecuación química balanceada. En este ejercicio se aplica el concepto de reactivo limitante, porque se proporcionan masas de dos reactivos y se debe identificar cuál restringe la cantidad de producto formado. Para ello, se convierten las masas a moles, se comparan las relaciones molares y se calcula la masa de producto a partir del reactivo limitante. El principio de conservación de la masa y la igualdad de la ecuación balanceada gobiernan el procedimiento.

### 2. Información proporcionada

Ecuación química balanceada:

\[
2\mathrm{H_2} + \mathrm{O_2} \longrightarrow 2\mathrm{H_2O}
\]

Valores de las masas de los reactivos:

\[
m_{\mathrm{H_2}} = 4\,\text{g}
\]

\[
m_{\mathrm{O_2}} = 32\,\text{g}
\]

Valores de las masas molares:

\[
M_{\mathrm{H_2}} = 2\,\text{g/mol}
\]

\[
M_{\mathrm{O_2}} = 32\,\text{g/mol}
\]

\[
M_{\mathrm{H_2O}} = 18\,\text{g/mol}
\]

### 3. Lo que se debe determinar

Se debe determinar la masa de agua formada a partir de las masas dadas de los reactivos.

\[
m_{\mathrm{H_2O}} = ?
\]

### 4. Relaciones químicas y expresiones necesarias

La ecuación balanceada indica las relaciones molares:

\[
2\text{ mol } \mathrm{H_2} \longrightarrow 2\text{ mol } \mathrm{H_2O}
\]

\[
1\text{ mol } \mathrm{O_2} \longrightarrow 2\text{ mol } \mathrm{H_2O}
\]

Para convertir masa a moles se usa:

\[
n = \frac{m}{M}
\]

Para identificar el reactivo limitante se comparan las relaciones molares:

\[
\frac{n_{\mathrm{H_2}}}{2} \quad \text{y} \quad \frac{n_{\mathrm{O_2}}}{1}
\]

Para calcular la masa de producto se usa:

\[
m_{\mathrm{H_2O}} = n_{\mathrm{H_2O}} \times M_{\mathrm{H_2O}}
\]

### 5. Aplicación de los datos

Se convierten las masas de los reactivos a moles:

\[
n_{\mathrm{H_2}} = \frac{4\,\text{g}}{2\,\text{g/mol}} = 2\,\text{mol}
\]

\[
n_{\mathrm{O_2}} = \frac{32\,\text{g}}{32\,\text{g/mol}} = 1\,\text{mol}
\]

Se comparan las relaciones molares para identificar el reactivo limitante:

\[
\frac{n_{\mathrm{H_2}}}{2} = \frac{2}{2} = 1
\]

\[
\frac{n_{\mathrm{O_2}}}{1} = \frac{1}{1} = 1
\]

Ambos valores son iguales, por lo tanto, los reactivos se consumen completamente y ninguno es en exceso.

### 6. Resolución paso a paso

Se calcula la cantidad de agua producida a partir de cada reactivo:

Para el hidrógeno:

\[
n_{\mathrm{H_2O}} = 2\,\text{mol } \mathrm{H_2} \times \frac{2\,\text{mol } \mathrm{H_2O}}{2\,\text{mol } \mathrm{H_2}} = 2\,\text{mol}
\]

Para el oxígeno:

\[
n_{\mathrm{H_2O}} = 1\,\text{mol } \mathrm{O_2} \times \frac{2\,\text{mol } \mathrm{H_2O}}{1\,\text{mol } \mathrm{O_2}} = 2\,\text{mol}
\]

Ambos cálculos coinciden, confirmando que los reactivos se consumen completamente.

Se calcula la masa de agua formada:

\[
m_{\mathrm{H_2O}} = 2\,\text{mol} \times 18\,\text{g/mol} = 36\,\text{g}
\]

### 7. Resultado obtenido

\[
\boxed{m_{\mathrm{H_2O}} = 36\,\text{g}}
\]

### 8. Conclusión

La masa de agua formada es \(36\,\text{g}\). Este resultado es coherente porque las masas de los reactivos se encuentran en la proporción estequiométrica exacta indicada por la ecuación balanceada, por lo que ninguno de los reactivos queda en exceso y
