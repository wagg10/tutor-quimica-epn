### 1. Fundamento químico del problema
Para determinar la masa de producto formado en una reacción química, primero se debe identificar el reactivo limitante, ya que este controla la cantidad máxima de producto que puede formarse. Una vez identificado, se utiliza la estequiometría de la ecuación balanceada para calcular la masa del producto.

### 2. Información proporcionada
Reacción química: \(2H_2 + O_2 \rightarrow 2H_2O\).

Masa de hidrógeno disponible: \(4 \, \text{g de } H_2\).

Masa de oxígeno disponible: \(32 \, \text{g de } O_2\).

Pesos atómicos: \(H = 1 \, \text{g/mol}\), \(O = 16 \, \text{g/mol}\).

### 3. Lo que se debe determinar
La masa de agua (\(H_2O\)) formada a partir de las masas dadas de \(H_2\) y \(O_2\).

### 4. Relaciones químicas y expresiones necesarias
Ecuación balanceada: \(2H_2 + O_2 \rightarrow 2H_2O\).

Masa molar del \(H_2\): \(M_{H_2} = 2 \, \text{g/mol}\).

Masa molar del \(O_2\): \(M_{O_2} = 32 \, \text{g/mol}\).

Masa molar del \(H_2O\): \(M_{H_2O} = 18 \, \text{g/mol}\).

Relación estequiométrica entre \(H_2\) y \(O_2\): \(2 \, \text{mol } H_2\) requieren \(1 \, \text{mol } O_2\).

### 5. Aplicación de los datos
Cálculo de los moles disponibles de cada reactivo:

\[
n_{H_2} = \frac{4 \, \text{g}}{2 \, \text{g/mol}} = 2 \, \text{mol}
\]

\[
n_{O_2} = \frac{32 \, \text{g}}{32 \, \text{g/mol}} = 1 \, \text{mol}
\]

### 6. Resolución paso a paso
**Paso 1: Identificar el reactivo limitante.**

Para que se forme agua, la relación estequiométrica debe cumplirse: \(2 \, \text{mol } H_2\) requieren \(1 \, \text{mol } O_2\).

Para los \(2 \, \text{mol}\) de \(H_2\) disponibles, se necesitan:

\[
n_{O_2} \text{ requeridos} = 2 \, \text{mol } H_2 \times \left(\frac{1 \, \text{mol } O_2}{2 \, \text{mol } H_2}\right) = 1 \, \text{mol } O_2
\]

Como se dispone exactamente de \(1 \, \text{mol}\) de \(O_2\), ambos reactivos se consumen completamente. En este caso, ambos pueden considerarse reactivos limitantes, pero para el cálculo del producto se puede usar cualquiera de ellos.

**Paso 2: Calcular la masa de agua formada.**

Usando los moles de \(H_2\):

\[
m_{H_2O} = n_{H_2} \times \left(\frac{2 \, \text{mol } H_2O}{2 \, \text{mol } H_2}\right) \times M_{H_2O}
\]

\[
m_{H_2O} = 2 \, \text{mol} \times 1 \times 18 \, \text{g/mol} = 36 \, \text{g}
\]

Verificación usando los moles de \(O_2\):

\[
m_{H_2O} = n_{O_2} \times \left(\frac{2 \, \text{mol } H_2O}{1 \, \text{mol } O_2}\right) \times M_{H_2O}
\]

\[
m_{H_2O} = 1 \, \text{mol} \times 2 \times 18 \, \text{g/mol} = 36 \, \text{g}
\]

Ambos métodos confirman que se forman \(36 \, \text{g}\) de agua.

### 7. Resultado obtenido

\[
\boxed{m_{H_2O} = 36 \, \text{g}}
\]

### 8. Conclusión
La masa de agua formada a partir de \(4 \, \text{g}\) de \(H_2\) y \(32 \, \text{g}\) de \(O_2\) es de \(36 \, \text{g}\). Este resultado es lógico porque ambos reactivos se consumen completamente y la relación estequiométrica de la reacción se cumple exactamente.
