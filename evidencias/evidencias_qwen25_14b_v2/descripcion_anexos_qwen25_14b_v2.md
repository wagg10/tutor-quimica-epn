# Descripción de anexos - Qwen2.5-14B-Instruct v2

Esta carpeta contiene las evidencias técnicas generadas durante el entrenamiento del modelo Qwen2.5-14B-Instruct adaptado mediante QLoRA para tutoría académica en Química.

Los anexos se organizaron de la siguiente manera:

- `01_script`: script de entrenamiento utilizado.
- `02_logs`: registros de ejecución del entrenamiento.
- `03_reportes_json`: reporte final, configuración e historial de métricas.
- `04_tablas_resultados`: tablas resumidas para la sección de resultados.
- `05_modelo_final`: evidencia del adaptador QLoRA generado.
- `06_checkpoints`: evidencia de checkpoints intermedios.
- `07_dataset`: evidencia del dataset final utilizado.
- `08_capturas_terminal`: información de GPU y versiones de librerías.
- `09_pruebas_inferencia`: espacio para guardar pruebas cualitativas posteriores.

Estas evidencias permiten respaldar la configuración experimental, el proceso de entrenamiento, las métricas obtenidas y los archivos generados por el modelo.
